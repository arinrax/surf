# Кулинарная студия "Шеф-стол" — Mock-бэкенд

## Описание
Mock-бэкенд для клиентского приложения кулинарной студии. Эмулирует работу REST API через Python-функции. Используется для демонстрации процесса работы с ИИ-агентами в рамках летней школы Surf.

## Технологии
- Python 3.x
- SQLite (стандартная библиотека `sqlite3`)
- Без внешних зависимостей

## Структура
- `db.py` — работа с базой данных
- `api.py` — mock API функции
- `client_testing.py` — тестовые сценарии
- `mock_data.json` — моковые данные

## Запуск
```bash
python client_testing.py
```

## Использованные ИИ-инструменты
- Qwen
- Промпты зафиксированы в `prompts_log.md`

## Артефакты
- `01_analytics.md` — доменная модель, User Stories
- `02_architecture.md` — схема БД, контракт API

- `04_bugs.md` — исправление 1-3 багов
- `05_qa.md` — ревью ТЗ, тест-кейсы
